# CMD-Using-Typescript-Vue
## 项目技术栈介绍：
- 编程语言：TypeScript4.x + JavaScript
- 构建工具：Webpack5.x
- 前端框架：Vue3.x + setup
- 路由工具：Vue Router 4.x
- 状态管理：Pinia
- UI框架：Element Plus
- 可视化：Echarts 5.x
- 工具库：@vueuse/core + dayjs + countup.js等
- CSS预编译：Sass/Less
- HTTP工具：axios
- Git Hook工具：husky
- 代码规范：EditorConfig + Prettier + ESLint
- 自动部署：Centos + Jenkins + Nginx
